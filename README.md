# Personal Task Management Backend

NestJS TypeScript API for personal task management.

## Installation

```bash
npm install
```

## Environment Variables

Create a `.env` file in the root directory:

```
PORT=3000
```

## Running the app

```bash
# development
npm run start:dev

# production mode
npm run build
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# test coverage
npm run test:cov
```

## Linting and Formatting

```bash
# lint
npm run lint

# format
npm run format
```
